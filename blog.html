Important FAQ

What is Document Object Model (DOM) ?

The DOM (Document Object Model) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as a tree of objects.

How can you select an element in the DOM using JavaScript?

You can select an element using methods like document.getElementById(), document.getElementsByClassName(), document.getElementsByTagName(), or document.querySelector().

What is the difference between getElementById() and querySelector()?

getElementById() selects a single element by its ID, while querySelector() can select any element that matches a CSS selector (including classes, IDs, or tag names) and returns the first matching element.

How do you change the text content of an element?

You can change the text content of an element using the textContent property.

What is event delegation in the context of the DOM?

Event delegation is a technique where a single event listener is added to a parent element instead of individual child elements. This way, you can manage events more efficiently and handle events for dynamically added elements.
